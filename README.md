### Описание

Доска задач, где можно создавать статусы и задачи. Также на сайте есть 
возможность зарегистрироваться. Задачи можно редактировать и перетаскивать между 
статусами.

---
### Стек:
- ReactJS
- TypeScript
- SCSS
- Redux
- Rest API
- Redux-Saga
- React Hook Form
- Jest
- React Router v6
---
